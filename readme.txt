QAColor
-------
by Timothy Bourke <timbob@bigpond.com>)

Derived from custom_tag.py (by Peter Bienstman <Peter.Bienstman@UGent.be>)
see also whiteonblack.py   (by Daniel Snider)

Plugin for Mnemosyne 1.1 (not 1.0.x!).

Adds a <color> tag with two optional attributes fg and bg that are used to
set the foreground and background colors respectively. Valid values are
listed at http://en.wikipedia.org/wiki/X11_color_names.

For example:
    <color fg="blue" bg="red">
